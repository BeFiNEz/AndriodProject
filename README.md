# AndriodProject
โปรเจคนี้ชื่อว่า FACEBOOKXcd เป็นการนำความรู้ที่เรียนมา มาปรับใช้ในการสร้าง
# องค์ประกอบที่เรานำมาพัฒนาแอปพลิเคชัน
- Fragment
- Recycler view
- Login Facebook Authen
- Realtime database (show, insert, update, delete)
- Dialog alert
- Graph by MPAndroidChart
- http
